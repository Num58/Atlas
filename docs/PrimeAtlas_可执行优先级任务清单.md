# PrimeAtlas 可执行优先级任务清单

> **文档定位**：基于完整功能蓝图（110+功能/79页面）的可执行开发任务清单
> **编制日期**：2026-07-02
> **编制人**：PrimeAtlas 项目经理 + 产品经理
> **核心原则**：不区分 MVP/Phase，所有任务按优先级 P0→P1→P2 排序，每个任务可独立开发、可测试、可验收

---

# 第一部分：任务拆解总表

## 1.0 阅读说明

- **任务ID**：T-XXX 格式，按建议开发顺序递增
- **功能ID**：对应蓝图中的功能ID（CP/TR/EN/RD/DA/MT/AG/SC/DP/CS/IF）
- **依赖任务**：被依赖的任务必须在当前任务开始前完成
- **负责角色**：前端=客户端（Android/iOS/小程序），后端=服务端+API，AI=AI 工程师，设计=UI/UX 设计师，测试=QA
- **预估工时**：基于全职投入，含设计+开发+联调+自测
