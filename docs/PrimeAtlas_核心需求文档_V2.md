# PrimeAtlas — 核心需求文档 V2.0

> **一句话定位：** 一个有灵魂的个人效能系统——AI训练伙伴陪你变强，不只是工具，更是陪伴。

---

## 一、产品灵魂

### 用户为什么需要 PrimeAtlas？

现在人普遍焦虑，做出改变的第一反应就是健身、学英语、读书。但现有的工具都是冷冰冰的——Keep 给你模板，多邻国给你关卡，微信读书给你书架。**没有一个产品让你感觉"有人在陪我变强"。**

PrimeAtlas 的核心差异化：**不只是 AI 工具，是一个有名字、有性格、有记忆、会成长的训练伙伴。** 它记得你的里程碑，注意到你的异常，在你状态差时给理解而非打鸡血，在你突破时比你还激动。

### 三大支柱

| 支柱 | 做什么 | 怎么与众不同 |
|------|--------|------------|
| 🏋️ **专项健身** | 多场景定制化训练计划 + 动态调整 | AI 不照搬书本，根据你的实际数据编排。早上家→中午单位→晚上球场，全天统筹 |
| 🗣️ **AI 英语听说** | 碎片化听说练习 + 场景融合 | 只做听说（最大痛点），组间休息也能练。健身场景英语深度融合 |
| 📚 **AI 读书伙伴** | 书籍 Agent 对话 + 跨域关联 | 不是被动读，是可对话、可关联训练计划的 Agent。"参考但不绝对" |

### 五大设计铁律

1. **"参考但不绝对"** — AI 提供分析推理，用户拥有最终决策权
2. **"场景融合"** — 不是三个 App 拼凑，是深度关联的有机整体
3. **"碎片化不是妥协"** — 15 分钟/5 分钟/2 分钟都是场景化设计
4. **"AI 透明推理"** — 展示推理过程，建立信任
5. **"数据自有"** — 用户数据归用户所有

## 三、核心功能

### 3.1 训练伙伴系统（产品灵魂）

**不是语音助手，是一个有"人设"的陪伴者：**

- **5 个初始伙伴**：Alex（硬核老兵）、小野（街头篮球）、Elena（运动科学博士）、K（数据狂人）、凛（禅修武者）
- **伙伴有记忆**：记住你的里程碑、注意到你的异常、记得你说过的话
- **伙伴会成长**：随训练次数解锁新能力（数据分析、故事模式、精准预判）
- **动态头像**：会呼吸、眨眼、随训练强度变化表情、PR 突破时跳起来
- **多模态陪伴**：语音有情绪变化、早安消息每天不同、视觉有存在感

**伙伴的关键时刻：**

| 场景 | 伙伴的响应 |
|------|-----------|
| 🟢 状态极佳 | "你今天状态特别好！要不要试试加一组挑战？" |
| 🟡 轻微疲劳 | "节奏不错，但不用太拼。把重量降 5%，质量比数量重要。" |
| 🟠 明显疲劳 | "我注意到你今天节奏比平时慢。要不要提前结束？已经足够了。" |
| 🔴 状态很差 | "今天能来就已经赢了。我准备了 15 分钟呼吸和拉伸，不算训练。" |
| 💔 用户说"不想练" | 不打鸡血，先共情："有时候就是不想动。我懂。" |

### 3.2 专项健身训练

- **多场景编排**：早上家(15min)→中午单位(10min)→晚上球场(75min)，全天统筹
- **专项目标定制**：篮球弹跳、羽毛球、游泳、减脂、产后康复等
- **动态调整**：基于 ACWR 负荷监控 + RPE 反馈 + 疼痛熔断
- **动作学习**：三层体系（关键要领→标杆视频→详细教学）
- **收益量化**：1RM 趋势、弹跳估算、专项能力指数、自动周报
- **专业保障**：11 个 Agent 组成的评审委员会交叉验证每个训练计划

### 3.3 AI 英语听说

- **只做听说**：瞄准最大痛点，不做读写单词
- **LISA 四维评估**：听力、发音可懂度、表达自发度、表达充分度
- **场景融合**：组间休息 30 秒英语对话、通勤跟读训练、健身术语学习
- **碎片化适配**：开车模式（纯语音）、通勤模式、睡前模式

### 3.4 AI 读书伙伴

- **万能资源摄入**：支持 PDF/视频/播客/图片/社交媒体/课程等 6 种形式
- **书籍 Agent 对话**：以书中专家口吻对话，四层深度（基础→关联→批判→应用）
- **跨域关联**：运动书籍→关联训练计划，英语书籍→关联学习路径
- **"参考但不绝对"**：资源信任度评估 + 冲突展示 + AI 保留意见

### 3.5 开放 Agent 生态

- **Agent 协作**：像专家团一样互相评审、借鉴、协作
- **Agent 商店**：厂商 Agent（OpenAI/DeepSeek）+ 三方 Agent（健身博主/康复机构）+ 用户自建
- **自然语言创建**：说"我要一个像科比一样严厉的篮球教练"→自动生成 Agent
- **产品代为决策**：专业术语内部消化，用户只看到自然语言 Tips

## 四、高能量交互体验（核心差异化）

### 4.1 训练中的仪式感

**3-2-1-GO 倒计时：**
- 最后 3 秒屏幕边缘出现能量光晕，从橙色→红色→金色
- GO! 瞬间数字碎裂成金色粒子飞散 + 闪白 + 伙伴喊"来！"

**最后一组：**
- 屏幕顶部滑入"⚠️ LAST SET"横幅 + 火焰粒子
- 手机震动 × 2 + 伙伴语音"把所有剩下的力气都放进去"

**PR 预警→突破：**
- AI 检测到可能破 PR → 全屏金色边框 + 史诗 BGM
- 确认突破 → 8 秒全屏庆祝：粒子爆发→"新纪录！"大字→数据对比滚动→伙伴跳起来
- 三种风格高光卡片可分享

### 4.2 伙伴的每日陪伴

**早安（每天不同）：**
> "早。阳光不错。昨天节奏不错，今天深蹲日。准备好了就开干。"

**PR 第二天：**
> "早。昨天那个深蹲 PR，我看了三遍回放。90kg，干净利落。"

**周末总结（伙伴的一封信）：**
> "这周你训练了 4 天。我知道周三那次是在加班之后来的。不是所有的胜利都写在数字里。有时候，来了就是胜利。"

### 4.3 智能情绪感知

- 检测 RPE 偏离、停顿频率、训练时长 → 判断用户状态
- 自适应响应：推一把（状态好时建议加量）vs 退一步（状态差时建议休息）
- 永远给用户选择权，不强制

## 五、数据反哺（隐私优先）

- **Opt-in 授权**：默认关闭，用户主动开启
- **四层脱敏**：PII 剥离→泛化→K-匿名化→差分隐私
- **贡献者权益**：青铜→白银→黄金→钻石，解锁不同权益
- **反哺路径**：脱敏数据→模式挖掘→Agent 优化→更好的推荐

## 六、MVP 规划

### Phase 1（3-4 个月）

```
✅ 训练伙伴系统（5 个初始伙伴 + 动态头像 + 基础记忆）
✅ 单场景训练计划 + 训练执行记录 + 动作视频
✅ AI 英语自由对话 + 跟读训练
✅ AI 读书（PDF 导入 + Agent 基础对话 + 笔记）
✅ 基础数据仪表盘
✅ 高能量交互（倒计时/PR 庆祝/最后一组仪式）
```

### Phase 2（+2-3 个月）

```
✅ 多场景全天编排 + ACWR 监控 + 疼痛熔断
✅ 跨域关联（运动书→训练计划）
✅ Agent 评审委员会
✅ 伙伴成长系统（Lv.2-3 解锁）
```

### Phase 3（+2-3 个月）

```
✅ 开放 Agent 生态 + Marketplace
✅ 小程序版本
✅ 社交功能（影子模式 + 碰拳）
✅ 数据反哺系统
```
